## üöÄ Getting Started

### Prerequisites

- Node.js (>= 12.x)
- npm or yarn

### Installation

1. Clone the repository
2. Navigate to the portfolio directory
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

### Production Build

```bash
npm run build
```

Creates an optimized production build in the `build` folder.

### Preview Production Build

```bash
npm run preview
```

Serves the production build locally for testing.

## üîß Customization

### Adding New Projects

Edit `src/Pages/Home/Projects.jsx` and add new project objects:

```javascript
{
  title: "Project Name",
  description: "Project description",
  technologies: ["React", "Node.js", "MongoDB"],
  link: "https://github.com/username/project"
}
```

### Updating Skills

Edit `src/Pages/Home/Skills.jsx` and modify the `skillsList` array.

### Styling

- Main styles: `src/App.css`
- Component-specific styles: Individual CSS files
- CSS Variables: Defined in `:root` for easy theming
